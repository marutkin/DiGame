# DiGame

Простенькая RPG в стиле классических RPG Maker игр, сделанная для смартфонов. Полностью playable в браузере и бесплатно хостится на GitHub Pages.

## Что это

- Top-down исследование мира на тайлах (как в старых RPG Maker 2000/2003)
- Диалоги с несколькими страницами и выборами (на русском)
- Grid-based движение (шаг за шагом)
- Карты редактируются в бесплатном Tiled
- Всё статическое — идеально для GitHub Pages

Текущая версия — playable короткая история:
- 2 карты (деревня + лес), соединённые варпами
- Диалоги с выборами и последствиями (на русском)
- Инвентарь, сохранения, простая пошаговая битва с Лесным духом
- Виртуальные контролы для смартфонов
- Полностью работает в браузере и на GitHub Pages.

## Быстрый старт (локально)

```bash
# 1. Установи зависимости (нужен Node.js 18+)
npm install

# 2. Запусти dev-сервер
npm run dev
```

Открой показанный в терминале URL (обычно http://localhost:5173).

**Для тестирования на телефоне**:
- Запусти `npm run dev`
- В терминале появится Network URL (например `http://192.168.1.42:5173`)
- Открой его на смартфоне в той же Wi-Fi сети

## Управление

- **Стрелки / WASD** — движение (шаг за шагом)
- **SPACE** — взаимодействие (говорить с NPC, активировать события, выбор в диалогах)
- **P** — быстрое сохранение
- На телефоне: автоматически появляются виртуальные кнопки (D-pad + большая кнопка A для действия). Полностью поддерживается тач.

## Сборка и публикация (GitHub Pages)

```bash
npm run build
```

Сборка попадает в папку `docs/`. 

### Как включить GitHub Pages (бесплатно):

1. Запушь репозиторий на GitHub.
2. Settings → Pages
3. Source: **Deploy from a branch**
4. Branch: `main` / Folder: `/docs`
5. Save

Через 1–2 минуты игра будет доступна по адресу:
`https://<твой-юзер>.github.io/<название-репо>/`

## Структура проекта (важно)

```
assets/
  maps/           # .json из Tiled (редактируй здесь карты)
  data/           # dialogues.json, items.json и т.д. (контент)
  images/         # tilesets, sprites (пока заглушки)
src/
  game/
    scenes/       # TitleScene, OverworldScene и т.д.
    systems/      # DialogueManager, EventSystem, SaveManager (в разработке)
  main.ts
vite.config.ts    # настроен на вывод в docs/ + base './' для Pages
SKILLS.md         # главная инструкция для разработчика и ИИ (читай обязательно)
AGENTS.md         # чтобы Grok Build автоматически подхватывал правила проекта
```
