# textjrpg-dasha

Короткая линейная мини-JRPG «Даша против венгров и Людмилы Конюховой».

Сделана специально под iPhone 16 Pro / мобильный Safari (iOS 18+).

## Технологии
- Phaser 3.80+
- TypeScript
- Vite
- Полностью автономный проект (все ассеты локальные)

## Запуск

```bash
cd textjrpg-dasha

# Установка зависимостей (первый раз)
npm install

# Запуск в режиме разработки
npm run dev
```

Открой http://localhost:5174 в Safari (или любом браузере).

## Сборка

```bash
npm run build
```

Готовые файлы будут в `dist/`.

## Управление

**На тач-устройствах (iPhone):**
- Виртуальный джойстик слева внизу
- Касайся NPC / триггеров для запуска диалога или боя

**На клавиатуре:**
- WASD или стрелки — движение
- Клик по экрану или кнопка "ВЗАИМ." — взаимодействие

## Сюжет (коротко)

1. Встреча с Димой → получаешь квест
2. Встреча с Плюшей (собака присоединяется к партии)
3. 1–2 боя с венграми на дороге
4. Финальный бой с Людмилой Конюховой
5. Юмористический эпилог

## Сохранения

Прогресс (квест + здоровье партии) сохраняется в `localStorage`.
Чтобы сбросить: удали ключ `dasha_quest_state` или нажми «НАЧАТЬ СНАЧАЛА» на экране победы.

## Структура проекта

См. изначальный запрос пользователя. Все сцены, системы и сущности разнесены по папкам.
